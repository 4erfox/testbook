ОПИСАНИЕ СТЕКА
==============

Frontend:
  - Vanilla JavaScript (без фреймворков для админки)
  - HTML5
  - CSS3
  - Vite (сборщик)
  - Marked (парсер markdown)

Backend:
  - Node.js
  - Express (REST API)
  - jsonwebtoken (JWT аутентификация)
  - bcryptjs (хеширование паролей)
  - WebSocket (для real-time коммуникации)

Хранилище:
  - Файловая система (markdown документы)
  - JSON файлы (конфиги, данные)


ОСНОВНОЙ ФУНКЦИОНАЛ
===================

1. Сайт-туториал:
   - Разделы с документами по деловому этикету
   - Поиск по документам
   - Переключение темы (светлая/темная)
   - Отзывчивый дизайн

2. Админ-панель:
   - JWT аутентификация (логин/пароль)
   - Редактирование документов (markdown + frontmatter)
   - Управление структурой навигации
   - Загрузка и управление файлами
   - Редактирование контактов
   - Система уведомлений
   - Переключение темы

3. Безопасность:
   - JWT токены для авторизации
   - Хеширование паролей (bcryptjs)
   - Валидация входных данных
   - CORS для API
   - Rate limiting


КОНФИГУРАЦИОННЫЕ ФАЙЛЫ
=======================

admin.env:
  JWT_SECRET=your-secret-key
  ADMIN_USERNAME=admin
  ADMIN_PASSWORD_HASH=bcrypt-hash
  JWT_EXPIRES_IN=7d

admin-config.json:
  {
    "title": "Admin Panel",
    "wsUrl": "ws://localhost:5174",
    "apiUrl": "http://localhost:5174/api"
  }

nav.json:
  Структура меню с разделами и документами

contacts.json:
  {
    "company": "...",
    "email": "...",
    "phone": "...",
    ...
  }


КОМАНДЫ ДЛЯ ЗАПУСКА
===================

npm install              - Установка зависимостей
npm run dev              - Запуск в режиме разработки (Vite + Express)
npm run build            - Сборка для production
npm run preview          - Просмотр prod версии
node server.js           - Запуск только Express сервера

Админ-панель доступна на: http://localhost:5174/admin/
Основной сайт: http://localhost:5174/
